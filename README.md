コピー元、コピー先フィールドとボタンを設置するスペースフィールドを指定し、ボタンを押した際に入力値をコピーする機能を追加するプラグインです。

コピー元、コピー先フィールドは入力不可フィールドを指定することもできるため、通常修正できない領域の変更も可能です。
